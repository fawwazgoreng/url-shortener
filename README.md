# Url-Shortener

> simple web shortener url built with **express** + **Redis** (Database)

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Containerization | Docker / Podman |
| Api | express (REST API) |
| Database | Redis |

---

## 📁 Project Structure

```
├── app/        # rest api
```

---

## ⚙️ Prerequisites

Make sure you have these installed:

- **Docker / podman**

---

### Build app

```bash
docker built your_app_app .
docker compose down (Opsional)
docker compose up --build
```

APP will be available at [http://localhost:3000](http://localhost:3000)
